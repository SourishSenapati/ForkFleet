# FORKFLEET: THE SOFTWARE EMPIRE MAP

ForkFleet transforms the complexity of open-source ecosystems into a navigable 3D Code Cosmos. Built on high-fidelity GPU physics and a cost-neutral static architecture, it allows developers to pilot through galaxies of repositories and map the evolution of human logic.

[LIVE MISSION DECK](https://sourishsenapati.github.io/ForkFleet/)

## MISSION OBJECTIVES

- SPATIAL INTELLIGENCE: Navigate the universe of 1,000+ repositories embedded in hierarchical clusters based on language and influence.
- PILOT MODE: Engage direct flight controls (WASD) to explore dependency star-systems and innovation pulsars.
- EVOLUTIONARY RADAR: Simulate the growth and decay of software eras using the temporal scrub-line.
- ZERO-COST SCALING: Engineered for viral traffic using a build-time universe generation pipeline and static CDN distribution.

## FLIGHT CONTROLS

- MISSION LAUNCH: Initial overview mission briefing.
- W / S: Main Thrusters (Forward / Backward).
- A / D: Lateral Translation (Left / Right).
- SHIFT / SPACE: Vertical Elevation.
- MOUSE: Tactical Look / Steering.
- CLICK: Initiate Deep-Analysis on target sectors.

## STRATEGIC ARCHITECTURE

ForkFleet is designed for 100% availability during high-traffic viral events:

1. OFFLINE UNIVERSE BUILDER: A Node.js engine pre-calculates the spatial graph, celestial classes, and intelligence metrics at build time.
2. REACT THREE FIBER ENGINE: High-performance instanced rendering handles 50,000+ stars and 1,000+ ships locally on the user's GPU.
3. STATIC REDUNDANCY: The entire application is exported as a static site, deployable to GitHub Pages and Vercel simultaneously.

## DEPLOYMENT TO GITHUB PAGES

This project includes an automated CI/CD pipeline for GitHub Pages:

1. FORK THE COMMAND: Fork this repository to your strategic intelligence center.
2. ENABLE DEPLOYMENT: Go to Settings > Pages and set the Source to "GitHub Actions".
3. MISSION START: Every push to the main branch triggers the `deploy.yml` workflow, which:
   - Installs dependencies.
   - Generates the latest Universe Snapshot.
   - Exports the Next.js mission deck.
   - Deploys the artifacts to your GitHub Pages endpoint.
